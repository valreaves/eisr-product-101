# 🎓 EISR Product Knowledge 101

**An interactive GitHub Skills course for GitHub EISRs (and any GitHub seller) to master product knowledge, pricing, and discovery.**

---

## 🚀 How to take this course

1. **[Fork this repository](https://github.com/valreaves/eisr-product-101/fork)**
2. Wait ~20 seconds after forking — a bot will open **Step 1** as an issue in your fork
3. Follow each step's instructions. Complete the exercise → commit → open a PR
4. The Skills bot checks your work, closes the step, and opens the next one
5. Repeat through all 10 modules
6. Finish with a **capstone customer scenario** — write a full disco + pitch flow

**Estimated time:** 3-4 hours (spread across a week if you'd like)

## 🎯 Learning Outcomes

By the end you can:
- Name every GitHub SKU, price, and key differentiator without notes
- Run a 30-min disco covering any product with confident follow-up talk tracks
- Position GitHub against Sonar, Snyk, ADO, GitLab, CodeRabbit, Cursor
- Explain UBB, AI Credits, cost centers, and chargeback to a CFO
- Map plays and motions from the H2 Revenue Plan to real customer conversations

---

## 🧠 How each module works

Every module has 3 parts:

1. **Read** — a short markdown lesson (5-10 min)
2. **Quiz** — fill in `quiz.md` with your answers, commit + push
3. **Practice** — write a mock disco flow in `practice.md`

The Skills bot auto-checks your quiz on each push. When correct, it advances you to the next step.

---

## 🛠️ For instructors / co-authors

Modules are markdown-only. To edit content:
- Update `modules/NN-name/README.md` for the lesson
- Update `modules/NN-name/quiz.md` (answer key in `.github/steps/NN-answers.md`)
- Update the workflow in `.github/workflows/step-progression.yml` to change grading
