## 🚀 Take the Full Course

1. **[Fork this repository](https://github.com/valreaves/eisr-product-101/fork)**
2. Wait ~20 seconds — the bot opens **Step 1** as an issue in your fork
3. Each module = **read → study flashcards → mark complete → commit → next step**
4. Modules 1-9 are flashcard-based (no quiz per module — self-paced study)
5. **Module 10 = comprehensive final quiz** (30 questions)
6. **Module 11 = capstone customer scenario**

**Estimated time:** 3-4 hours total, or one module per day for two weeks.

## 🎯 Learning Outcomes

By the end you can:
- Name every GitHub SKU, price, and key differentiator without notes
- Run a 30-min disco covering any product with confident follow-up talk tracks
- Position GitHub against Sonar, Snyk, ADO, GitLab, CodeRabbit, Cursor
- Explain UBB, AI Credits, cost centers, and chargeback to a CFO
- Map plays and motions from the H2 Revenue Plan to real customer conversations

---

## 🧠 How each module works

**Modules 1-9 (Flashcards):**
1. **Read** — short lesson (5-15 min)
2. **Study flashcards** — [open the flashcard app](https://valreaves.github.io/eisr-product-101/flashcards/), filter to the module's tag
3. **Mark complete** — check the box in `complete.md`, commit + push. Bot advances you.

**Module 10 (Final Quiz):**
- Answer all 30 questions in `quiz.md` without peeking. Commit to advance.

**Module 11 (Capstone):**
- Write a full pitch plan for a mock customer (Acme Corp). Self-graded.
