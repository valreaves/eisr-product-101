# Step 3: Copilot Enterprise (the upsell)

**⏱️ ~15 min** · Goal: know when + how to upsell from Business to Enterprise.

## Read

**What's added over Business:**
- **Knowledge bases** — Copilot indexes your docs / wiki / Confluence, answers cite YOUR content
- **Cost centers** — real chargeback with separate Azure sub IDs
- **PR summaries** — auto-generated PR descriptions
- **Custom / fine-tuned models** — trained on your codebase
- **AI Credits** — $39/mo (2x Business)

**Price:** $39/user/mo

| Feature | Business | Enterprise |
|---|---|---|
| Code completion (FREE) | ✅ | ✅ |
| Chat / agent / CCR / Coding Agent | ✅ | ✅ |
| Knowledge bases | ❌ | ✅ |
| PR summaries | ❌ | ✅ |
| Custom / fine-tuned models | ❌ | ✅ |
| Cost centers | Limited | ✅ Full |
| AI Credits | $19 | $39 |

**Upsell triggers:** Chargeback need · Scattered internal docs · >500 seats · Wants fine-tuning

## Disco

| Question | Signal | Follow-up |
|---|---|---|
| "Devs waste time searching internal docs?" | "Yes, Confluence mess" | "Knowledge bases index all of it. Devs ask 'how does our auth work?' and get an answer citing YOUR docs. #1 differentiator." |
| "Chargeback by team?" | "Yes" | "Cost centers = Enterprise only. Separate Azure sub IDs. Real billing separation, not just reporting." |
| ">500 Copilot seats?" | "Yes, org-wide" | "At scale, Business gets fragile — need PR summaries + cost centers. $20 delta pays for itself in reviewer time." |
