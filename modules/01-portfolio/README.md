# Step 1: Portfolio Overview

**⏱️ ~10 min** · Goal: memorize every SKU, price, and one-liner.

## Read

GitHub sells the following products. Learn the price and one-liner for each.

| Product | List Price | One-liner |
|---|---|---|
| **GitHub Enterprise Cloud (GHEC)** | $21/user/mo | The platform: repos, Actions, Packages, admin |
| **GitHub Enterprise Server (GHES)** | $21/user/mo | Self-hosted GHEC for regulated / air-gapped |
| **Copilot Business** | $19/user/mo | AI code completion + chat + CCR + Coding Agent |
| **Copilot Enterprise** | $39/user/mo | Business + knowledge bases + PR summaries + custom models |
| **Copilot Code Review (CCR)** | Included | AI PR reviewer, agentic GA Feb 2026 |
| **Copilot Coding Agent** | Included | Autonomous agent — assign issues, get PRs |
| **GHAS (Bundled)** | $49/active committer/mo | Code Security + Secret Protection |
| **Code Security** (standalone) | $30/active committer/mo | CodeQL SAST + Dependabot + Autofix |
| **Secret Protection** (standalone) | $19/active committer/mo | Secret scanning + push protection |
| **Code Quality** (GA 7/20/26) | TBD | Hybrid AI+rules — displaces Sonar / Coverity |
| **GitHub Actions** | Included + overage | CI/CD, runs CCR + Coding Agent |
| **GitHub Packages** | Included + overage | npm/Maven/NuGet/Docker registry |

## Key concepts

- **Active committer** = anyone who pushed to a GHAS-enabled repo in last 90 days
- **AI Credit** = 1 credit = $0.01 USD. Copilot Business includes $19/mo, Enterprise $39/mo
- **What's FREE in Copilot** = code completion + Next Edit suggestions (never consume credits)
