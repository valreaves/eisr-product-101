# Step 8: Code Quality

**⏱️ ~10 min** · Goal: Sonar displacement + timing.

## Read

**Status:** Public Preview now → **GA July 20, 2026**. Standalone SKU (price TBD at GA).

**What it is:** Hybrid AI + rules code quality scanning. Native to the same code scanning platform as GHAS (quality lens vs security lens).

**Why customers pick it:**
- Native — same PR flow as GHAS findings, no separate dashboard
- Hybrid AI + rules — legacy tools rules-only, AI-native lack governance
- Enterprise reporting + policy enforcement
- Displaces Sonar / Coverity / Codacy / Parasoft
- AI code quality is THE risk right now
