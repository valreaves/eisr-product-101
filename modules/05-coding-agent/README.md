# Step 5: Copilot Coding Agent

**⏱️ ~10 min** · Goal: position the agentic story.

## Read

**What it is:** Autonomous agent — assign a GitHub issue to Copilot → it explores the repo → drafts a PR → you review + merge. Runs on GitHub Actions.

**Pricing:** Included with every paid Copilot license. Uses AI Credits per task + Actions minutes.

**Best for:** The "toil pile" — bugs, tests, docs, refactors, boilerplate.

## Disco

| Question | Signal | Follow-up |
|---|---|---|
| "Backlog of small bugs / tests / docs?" | "Huge" | "That's what Coding Agent absorbs. Assign like a teammate — it drafts a PR. Your seniors review + merge." |
| "% toil vs feature work?" | ">30% toil" | "Toil is what the agent is built for. Output flows into your normal PR → CCR → merge. No parallel workflow." |
| "Evaluating Devin / Cursor?" | "Yes" | "Coding Agent runs on YOUR Actions, respects YOUR runners, YOUR governance. Included in Copilot — no separate contract." |
